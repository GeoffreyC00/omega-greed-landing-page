# Omega Greed Landing Page

Dedicated email opt-in landing page project for Omega Greed.

## Purpose

This repo keeps a saved, versioned copy of the landing-page direction, copy, and prototype so changes can be reviewed, reverted, and pushed to GitHub.

## Landing Page Goal

Send direct traffic to one page with one primary action:

**Join the Omega Greed email list.**

The page should keep the tone of the main website while focusing on cinematic motorcycle storytelling, night rides, LA atmosphere, culture, horror, and adventure.

## Squarespace Build Notes

Recommended setup:

1. Create a blank Squarespace page at `/join` or `/newsletter`.
2. Keep the site header/footer or use a simplified header with the Omega Greed logo.
3. Use the current Squarespace form block connected to the existing Google Drive spreadsheet.
4. Use this repo's prototype and copy as the design reference.
5. Keep the page focused on the form; avoid competing calls to action above the fold.
